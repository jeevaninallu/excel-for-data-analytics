# Excel Practice

This folder contains my Excel practice files while learning Excel for Data Analytics.

## Topics Covered

### Excel_1_Basics

In this file I learned:

- Workbook and Worksheet
- Rows, Columns, Cells
- Sheet Tabs
- Ribbon and Toolbar
- Cursor types
- Basic shortcuts
- Default alignment
- Data entry trick
- Introduction to formulas
- Basic formulas

---

### Excel_2_data_jobs_salary_monthly

In this file I learned copying data from one sheet to another in the same workbook using different methods:

- Copy using Ctrl + C and Ctrl + V
- Copy using formula (=Sheet1!A1)
- Copy using Move or Copy option

---

### Excel_3_Workbooks

In this file I learned copying data between different workbooks:

- Copy using Ctrl + C and Ctrl + V (different workbook)
- Copy using formula between workbooks
- Copy using Move or Copy between workbooks

---
